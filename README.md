# disaster-tweets-ml-classifiers
Machine learning project for classifying disaster-related tweets. Includes: data visualizations to understand the target, feature engineering (statistics, TF-IDF and BERT embeddings, NER + geodecoding), various ML models (LogisticRegression, KNN, XGBoost RandomForest). Also did ROC-curve, feat. importance, confusion matrix and a UMAP 3D reduction.
